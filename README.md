# slotlist.info

**Status:** Discontinued as of December 31, 2025

## About

slotlist.info was an ArmA 3 mission planning and slotlist management tool that allowed communities to create and manage missions with registration systems for player roles. The service ran for nearly a decade before being shut down at the end of 2025.

## Features (Historical)

- Mission creation and management
- Slotlist system for role registration
- Community profiles and representation
- Steam Single Sign-On integration

## Shutdown Notice

This service has been permanently discontinued. The infrastructure and codebase became outdated and required modernization that was no longer feasible to maintain.

## Source Code

The source code remains available in this organization as an archive for historical reference and potential future use by others in the ArmA community.
